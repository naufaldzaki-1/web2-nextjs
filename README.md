# Web2 Next.js – Profil Singkat

Project ini dibuat sebagai bagian dari tugas mata kuliah **Pemrograman Web II**.
Website ini dibuat menggunakan **Next.js** dan menampilkan beberapa halaman sederhana seperti Home, Profile, dan About.

---

## 🚀 Cara Menjalankan Project

1. Clone repository ini

```
git clone https://github.com/naufaldzaki-1/web2-nextjs.git
```

2. Masuk ke folder project

```
cd web2-nextjs/create-next-app
```

3. Install dependencies

```
npm install
```

4. Jalankan development server

```
npm run dev
```

5. Buka di browser

```
http://localhost:3000
```

---

## 📂 Struktur Folder

```
app/
 ├ page.tsx          → Halaman Home
 ├ about/
 │   └ page.tsx      → Halaman About
 └ profile/
     └ page.tsx      → Halaman Profile

public/              → Asset seperti gambar
```

---

## 📄 Halaman Website

- **Home (`/`)**
  Halaman utama website yang berisi navigasi menuju halaman profile.

- **Profile (`/profile`)**
  Menampilkan informasi profil seperti nama, NIM, dan minat.

- **About (`/about`)**
  Berisi penjelasan tentang project website ini.
